# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/lucaslauriano/survivors-challange.git
```

```bash
$ cd /backend
# Install Dependencies
$ yarn
```

```bash
$ yarn start
```

```bash
$ cd ..
$ cd /frontend
# Install Dependencies
$ yarn
```

```bash
$ yarn start
```

Go to http://localhost:3000/login to see the result.
